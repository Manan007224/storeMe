# StoreMe

## To run the project, type: `npm run start`

API calls:

`/createCourses` **(POST)**
Create new course


`/COURSES/:id` **(GET, POST)**
Get meta information for course with `:id`


`/COURSES` **(GET, POST)**
Get list of all courses


`/COURSES/:id/deleteCourses` **(POST)**
Delete course with `:id`
